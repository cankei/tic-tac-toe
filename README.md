# tic-tac-toe

A basic Tic-tac-toe written with AngularJs.

Play it online:

https://tic-tac-toe-dc865.firebaseapp.com

### Get started
Install dependencies: `npm install`

Run unit tests: `npm test`

Open the app in browser: `npm start`
